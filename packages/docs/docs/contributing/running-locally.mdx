# 在本地运行

这是一个指南，将向您展示如何在本地运行此项目，以便测试或为我们的 Fuel Wallet SDK 做出贡献。

## 运行项目

### 要求

此项目包括前端。开始之前，请安装依赖项：

- [Node.js 18.14.1 或更高版本](https://nodejs.org/en/)。我们建议使用 [nvm](https://github.com/nvm-sh/nvm) 进行安装。
- [pnpm v7.18.2 或更高版本](https://pnpm.io/installation/)
- [Docker v0.8.2 或更高版本](https://docs.docker.com/get-docker/)
- [Docker Compose v2.6.0 或更高版本](https://docs.docker.com/get-docker/)

### 获取代码库

1. 访问 [Fuel Wallet](https://github.com/FuelLabs/fuels-wallet) 仓库并 fork 项目。
2. 然后将您 fork 的副本克隆到本地计算机并开始工作。

```sh
git clone https://github.com/FuelLabs/fuels-wallet
cd fuels-wallet
```

### 安装依赖项

```sh
pnpm install
```

### 设置环境变量

```sh
cp packages/app/.env.example packages/app/.env
```

### 运行本地节点

在此步骤中，我们将：

- 启动本地 `fuel-core` 节点；
- 启动本地 `faucet` API；

在运行以下命令之前，请确保已安装并运行 Docker。

```sh
pnpm node:up
```

### 运行 Web 应用

启动本地开发前端。运行以下命令后，您可以在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看前端。

```sh
pnpm dev
```

## 项目概述

此部分简要描述了每个目录。更多详细信息可以通过点击链接查看每个包中的内容。

- [packages/app](@repository/packages/app/) 前端 Fuel Wallet 应用程序
- [packages/config](@repository/packages/config/) 构建配置

### 有用的脚本

为了让生活更轻松，我们在 [`package.json`](@repository/package.json) 中添加了尽可能多的有用脚本。以下是开发过程中最常用的一些：

```sh
pnpm <command name>
```

| 脚本           | 描述                                                                         |
| -------------- | ---------------------------------------------------------------------------- |
| `dev`          | 运行用于 Web 应用程序的开发服务器 [packages/app](@repository/packages/app/)    |
| `dev:storybook`| 运行 Storybook，这是我们用来开发组件的地方。                                  |
| `test`         | 运行基于 Jest 的所有单元测试。                                                |
| `test:e2e`     | 运行基于 Cypress 的所有端到端测试。                                           |

> 其他脚本可以在 [`package.json`](@repository/package.json) 中找到。

### 运行测试

请确保首先完成以下步骤：

- [获取代码库](#获取代码库)
- [安装依赖项](#安装依赖项)
- [运行本地节点](#运行本地节点)

### 在开发模式下运行测试

所有测试都针对在文件 `packages/app/.env` 中配置的本地节点运行（如果存在文件 `packages/app/.env.example`）。

在运行测试之前，请确保本地测试节点正在运行：

```sh
pnpm node:up
```

然后，运行以下命令来运行测试：

```sh
pnpm test
```

### 运行端到端测试

要运行端到端测试，请按照以下步骤操作：

#### 在开发模式下运行端到端测试

```sh
pnpm node:up
```

```sh
pnpm dev:crx
```

```sh
pnpm test:e2e
```

#### 在 CI/TEST env 模式下运行端到端测试

```sh
pnpm node:up
```

```sh
NODE_ENV=test pnpm build:app
```

```sh
NODE_ENV=test pnpm test:e2e
```
