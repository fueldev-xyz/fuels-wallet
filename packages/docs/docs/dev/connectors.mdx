# Fuel钱包连接器

Fuel钱包连接器是由钱包开发人员提供的接口，允许您的DApp集成特定钱包。

您可以通过阅读[Fuel钱包连接器](https://github.com/FuelLabs/fuels-wallet/wiki/Fuel-Wallet-Connectors)规范来了解Fuel钱包连接器的工作原理。

## 列出连接器

在使用多个连接器时，您应该允许用户选择他们希望与您的DApp进行交互的连接器。一旦调用`connectors()`方法，Fuel钱包SDK将从连接器中查询信息，让您确定安装了哪些连接器。

我们还建议在使用`connectors()`方法的地方使用`connectors`监听器，因为可用性可能会发生变化。
