## 从源代码安装

您还可以直接从我们的源代码构建并安装Fuel Wallet的开发版本。步骤如下：

- 下载 <DownloadWalletZip />;
- 在Chrome或Brave中：
  - 打开扩展页面，可以通过以下方式完成：
    - 点击设置 -> 扩展程序 或;
    - 访问 `brave://extensions/` 或 `chrome://extensions/`。
  - 启用右上角的“开发者模式”开关。
  - 加载 `fuel-wallet.zip` 文件，可以通过：
    - 将下载的Fuel Wallet文件拖放到扩展页面中 或;
    - 点击“加载已解压的扩展程序”并选择该文件。
- 如果一切顺利，一个引导页面将会立即打开。

Fuel Wallet扩展程序现在已准备就绪。
