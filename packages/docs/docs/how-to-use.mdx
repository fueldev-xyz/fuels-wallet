# 使用说明

## 账户

### 查看所有账户

您可以通过点击账户图标右侧的“更改”按钮，或点击导航侧栏中的“账户管理”来查看所有账户。

/>

### 添加新账户

您可以通过点击账户页面上的“添加新账户”或“从私钥添加”按钮来添加新账户。

  media={[
    {

### 编辑账户

要编辑账户，请点击账户右侧的水平省略号并选择“编辑”。

  media={[
    {

要隐藏账户，请点击账户右侧的水平省略号并选择“隐藏账户”。请注意，您必须选择其他账户作为活动账户。

## 资产

### 发送和接收资产

点击钱包主屏幕上的“发送”按钮将任何资产转移到另一个地址。
点击“接收”按钮查看您的钱包地址及其二维码以从另一个地址接收资产。

  media={[

  ]}
/>

### 查看资产

您可以通过点击侧栏导航中“设置”下的“资产”查看所有自定义和列出的资产。

您可以通过点击“添加资产”按钮添加新的自定义资产。
您必须指定资产 ID、名称、符号和图像 URL。
如果您的钱包中已有未知资产，您可以点击“添加”按钮将其添加。
